Pipes

  • Built-in Pipes

    • Some Built-in pipes include lowercase formatting, uppercase formatting and the date formatting.


    Before Pipe_Name After
    {{ lowerCaseExample }} lowercase {{ lowerCaseExample | lowercase }}
    {{ upperCaseExample }} uppercase {{ upperCaseExample | uppercase }}
    {{ dateExample }} date {{ dateExample | date }}
    {{ userObject }} json {{ userObject | json }}
    {{ currencyExample }} currency {{ currencyExample | currency }}
    {{ percentExample }} percent {{ percentExample | percent }}

    Printing json objects is commonly used for debugging purposes:
    {{ userObject | json }}

  • Passing Parameteres in pipes

    • {{ dateExample | date: "dd/MMM-yy" }}

    from {{ percentExample }} transform to {{ percentExample | percent: "2.2" }}

  • Chaining in pipes

    • You can use multiple pipes on the same input as well!!

    For example using uppercase AND the DATE formatting:
    {{ dateExample | date | uppercase }}
